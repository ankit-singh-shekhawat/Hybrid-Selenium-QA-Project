# Manual UI/UX Test Cases - Swag Labs

### Test Case 1: Responsive Visual Layout
* **Objective:** Verify the store layout looks clean on standard screens.
* **Steps:**
    1. Open `https://saucedemo.com` and log in.
    2. Scan the product images.
Expected Manual Check: Images must not be broken, alignment must be straight, and text must be completely readable.

### Test Case 2: Cart Dynamic Visual Indicator
* **Objective:** Ensure the cart badge visually changes color and pops out when an item is added.
* **Expected Manual Check:** The red badge color must clearly contrast against the dark shopping cart icon for high user visibility.
